Selected work

The tools behind agentic payments on Polygon.

  1. 01 Polygon Agent CLI One install gives an AI agent a session wallet, an onchain identity, stablecoin payments, swaps, bridging, and pay-per-call API access — no private keys exposed. x402ERC-8004Polygon TrailsSession wallets
  2. 02 Polygon Facilitator Polygon's x402 facilitator (x402-rs) — the engine that verifies and settles every payment. Server middleware gates a route behind a price; client middleware pays it transparently. Production Rust. x402 v1 + v2AxumEVM
  3. 03 Agentic Services A directory of x402 services an agent can find and call, paying per request in USDC with no keys. Part of Polygon's Open Money Stack. x402USDCOpen Money Stack
  4. 04 AgentConnect Links an agent to a real, user-approved Polygon wallet, and serves the machine-readable skill agents fetch to learn the full onchain lifecycle. SKILL.mdSequencex402
  5. 05 Agentic docs The developer guide for agents that pay onchain — how they negotiate price, sign intents, and settle per request with x402 and ERC-8004. x402ERC-8004MCP
  6. 06 PIP-82 — Gas Program Co-authored Polygon Improvement Proposal — the Agentic Commerce Gas Program — recycling up to $1M in base-fee rebates to public x402 facilitators to cut gas friction for agent-to-agent commerce. x402PIPGas rebates

    About

    I'm a developer relations engineer at Polygon, where I lead our agentic payments work. I take a forgotten HTTP status code and turn it into something developers can ship: wallets, SDKs, and settlement that AI agents actually use.

    I joined as an intern; nine months later it was full-time. I build in the open — mostly where agents, payments, and cross-chain meet — and I write the proposals and tooling that try to make all three less painful.

    Long-term goal: make agents self-sufficient enough to do the work themselves. Then I finally get to nap.
