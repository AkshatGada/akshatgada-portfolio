Selected work

Recent work building the agentic payments stack at Polygon.

  1. 01 Polygon Agent CLI One install gives an AI agent a session wallet, an onchain identity, stablecoin payments, swaps, bridging, and pay-per-call API access — no private keys exposed. x402ERC-8004Polygon TrailsSession wallets
  2. 02 Agentic Services A directory of x402 services an agent can find and call, paying per request in USDC with no keys. Part of Polygon's Open Money Stack. x402USDCOpen Money Stack
  3. 03 AgentConnect Links an agent to a real, user-approved Polygon wallet, and serves the machine-readable skill agents fetch to learn the full onchain lifecycle. SKILL.mdSequencex402
  4. 04 Agentic docs The developer guide for agents that pay onchain — how they negotiate price, sign intents, and settle per request with x402 and ERC-8004. x402ERC-8004MCP
  5. 05 PIP-82 — Gas Program Co-authored Polygon Improvement Proposal — the Agentic Commerce Gas Program — recycling up to $1M in base-fee rebates to public x402 facilitators to cut gas friction for agent-to-agent commerce. x402PIPGas rebates
  6. 06 Polygon Facilitator Polygon's x402 facilitator (x402-rs) — the engine that verifies and settles every payment. Server middleware gates a route behind a price; client middleware pays it transparently. Production Rust. x402 v1 + v2AxumEVM

    Impact at Polygon

    The developer and ecosystem layer behind 24M+ x402 transactions.

    I joined when agentic payments on Polygon were still early and helped turn them into a developer-facing product surface. I led work across the Agent CLI, Agentic Services, and AgentConnect; brought pay-per-use services into Polygon's discovery layer; and worked directly with ecosystem teams on integrations, documentation, demos, and launches.

    The result was a clearer path for developers to build agents that can hold funds, discover services, and pay over x402—with the tools, integrations, and distribution needed to bring those services live on Polygon.
